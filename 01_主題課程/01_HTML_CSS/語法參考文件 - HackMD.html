      Flex

      宣告

      


      .container{
    display: flex;

      .container{
      flex-direction: …;
      flex-wrap: …;

      align-item: …;

      }

      flex-direction

      決定主軸方向

      






          flex-direction: row;
    /*
        row: 向右為主軸正向
        row-reverse: 向左為主軸正向
        column: 向下為主軸正向
        column-reverse: 向上為主軸正向
    */

      flex-wrap

      決定換行規則

      





          flex-wrap: wrap;
    /*
        預設：不換行，擠不下的自動縮放到塞的下
        wrap: 向下換行
        wrap-reverse: 向上換行
    */

      justify-content

      決定元素在主軸的位置

      







          justify-content: flex-start;
    /*
        flex-start: 從主軸負向開始往正向排
        flex-end: 從主軸正向開始往負向排
        center: 從中間開始排
        space-around: 平均分配寬度（頭尾計入）
        space-between: 平均分配寬度（頭尾不計入）
    */

      align-item

      決定元素在交錯軸的起始位置

      





          align-item: flex-start;
    /*
        flex-start: 交錯軸負向為第一列
        flex-end: 交錯軸正向為第一列
        stretch: 自動拉伸到每一個元素一樣高
    */

      align-content

      決定元素交錯軸的排列規則

      









          align-content: stretch;
    /*
        在 flex-wrap 被設定且不是預設值才有用
        flex-start: 往交錯軸正向排列
        flex-end: 往交錯軸負向排列
        center: 所有元素擠在中間
        stretch: 平均分配欄高（剩餘的空間會分配給每一行，且頭尾計入）
        space-around: 平均分配欄高（剩餘的空間不會分配給每一行，頭尾計入）
        space-between: 平均分配欄高（剩餘的空間不會分配給每一行，頭尾也不計入）
    */
